# PetShop Cão Feliz - Sistema Web (Fase 1)

## Funcionalidades da Página Web
Este projeto foi desenvolvido utilizando HTML puro para a disciplina de Fundamentos de Sistemas Web. A estrutura do sistema contempla as seguintes páginas e funções:

1. **index.html (Página Inicial):** Apresenta as boas-vindas ao sistema e o cabeçalho/rodapé padrão com links de navegação para as demais áreas do site.
2. **produtos.html (Catálogo):** Exibe 3 categorias de produtos (Rações, Brinquedos e Higiene). Cada categoria contém 2 produtos em destaque. Cada produto apresenta uma foto (imagem de placeholder), descrição textual e o valor do item.
3. **servicos.html (Serviços):** Lista os serviços do estabelecimento em formato de tópicos, contendo a descrição do serviço, o valor cobrado e a informação explícita se o serviço possui Tele-busca ou não.

Todos os arquivos possuem tags semânticas (<header>, <main>, <section>, <nav>, <footer>) garantindo a organização exigida para a fase 1, sem a utilização de CSS ou JS.
